To play the game, run the file titled main.py
Changes to the game can be made in the file config.py
the controls for the game are:
Player 1:
    Arrow left: rotate left
    Arrow right: rotate right
    Arrow up: thrust
    Right shift: shoot

Player 2:
    a: rotate left
    d: rotate right
    w: thrust
    Left shift: shoot
